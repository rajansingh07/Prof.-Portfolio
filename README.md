Welcome to the Portfolio repository. This is a professional portfolio website built to showcase academic achievements, research, projects, and professional journey as a Professor. The website is designed to be modern, responsive, and user-friendly, providing an easy way for students, colleagues, and collaborators to explore my work.

About the Project:
This portfolio highlights academic background, teaching experience, research contributions, and professional projects. It is developed using modern frontend technologies to provide a smooth and engaging experience across all devices. The website also provides quick access to resume, research publications, and contact information.

Features:
- Fully responsive design for desktops, tablets, and mobile devices
- Sections showcasing research, publications, and academic projects
- Highlighted skills and areas of expertise
- Downloadable CV/Resume for professional references
- Links to social and professional profiles for networking
- Clean, modern, and intuitive user interface

This portfolio is built using the following technologies:
- React.js for building dynamic and reusable UI components
- Vite as the frontend build tool for fast development
- Tailwind CSS for utility-first and responsive styling
- JavaScript for interactivity and animations
- HTML & CSS for structural markup and base styling
- Vercel / GitHub Pages for deployment

Getting Started

To run this portfolio locally:
Clone the repository

- git clone https://github.com/rajansingh07/Proff.-Portfolio.git

Navigate into the project folder:
- cd Proff.-Portfolio
- Install dependencies
- npm install
- Start the development server
- npm run dev
- Open your browser at:
- http://localhost:5173
